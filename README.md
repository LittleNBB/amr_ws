
#   项目：SLAM建图与自主控制小车

##   项目环境

- **操作系统**: Ubuntu 22.04 LTS
- **ROS 版本**: ROS 2 Humble (Desktop Full)
- **仿真器**: Gazebo 11 / RViz2
- **主要依赖**: `robot_state_publisher`, `xacro`, `slam_toolbox`, `nav2`

##   项目目的：

- 熟练git的使用
- 熟练使用ROS控制小车的行动
- 学会使用 URDF/Xacro 进行机器人3d建模
- 学会使用 Gazebo 建立3d物理世界进行模拟
- 学会使用 rviz 实现可视化
- 学会使用 SLAM 构建地图
- 学会使用 Nav2 进行自主导航

##   实现过程

- [] 机器人模型构建
- [] Gazebo 物理仿真与差速控制实现
- [] SLAM 扫描建图
- [] Nav2 自主导航
